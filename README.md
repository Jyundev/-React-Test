# 🚀 DDaja-it -frontend

따자잇(DDaja-it)은 it 업계 취업을 준비하며 it 관련 자격증 취득을 준비하고 있는 분들에게 관련 정보를 제공함과 동시에 자격증 취득을 위한 챌린지 프로그램을 제공합니다.


## 🔆사용 프레임워크

React + Vite


## 🛒 사용 npm 라이브러리

nodejs의 npm(node package manage)에서 사용한 라이브러리 목록입니다.

-react-router-dom
-react-hook-form
-styled-components
-prop-types
-styled-reset
-zustand
-axios
-aws-sdk
-fullcalendar
-react-table
-react-icons
